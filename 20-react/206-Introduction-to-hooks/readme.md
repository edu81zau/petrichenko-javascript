# Лекция 164. useState.

Общие правила работы с хуками.  
1. Хуки стоит вызывать только на верхнем уровне.   
 Не вызывать внутри циклов, условий или вложенных функций.  
2. Хуки стоит вызывать только из функциональных компонентов React
