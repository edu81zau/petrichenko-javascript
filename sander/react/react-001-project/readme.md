### useState — это хук React, который позволяет вам добавлять состояние
### в функциональные компоненты. Он является основным инструментом для
### управления данными, которые могут изменяться со временем и вызывать
### обновление пользовательского интерфейса.

## Основные принципы

* Инкапсуляция состояния: useState возвращает массив из двух элементов:

  Текущее значение состояния.

  Функцию для обновления этого значения.

* Иммутабельность: При обновлении состояния вы всегда должны создавать
  новое значение вместо того, чтобы мутировать старое. Например, для
  массива нужно использовать ... (спред-оператор) или .map(),
  а для объектов — ... или Object.assign(). Это гарантирует, что React
  обнаружит изменение и вызовет повторный рендеринг.

* Начальное значение: При вызове useState вы передаёте ему начальное
  значение. Если это ресурсоёмкая операция (например, вызов new Date()),
  можно передать функцию, которая вернёт это значение. React вызовет её
  только один раз при первом рендере.
