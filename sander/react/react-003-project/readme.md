## Здесь демонстрируется ключевой принцип React:
## разделение ответственности и повторное использование компонентов. 

* Используется декларативный подход
* Описывается, что хотим получить, а React делает всё остальное.
* Создаётся компонент Profile, который декларирует, как должен выглядеть один профиль.
* Затем в компоненте Gallery декларируется, какие данные нужно передать в Profile для каждого из учёных.
* Такой код гораздо чище, проще для чтения и легче для поддержки.
