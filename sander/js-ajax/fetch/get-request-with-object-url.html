Замечания к тесту

Расшифровка по компонентам URL
    https: - протокол
    site.com - hostname
    8080 - port
    /path/page - pathname
    ?p1=v1&p2=v2 - search
    #hash - hash
Кодирование

    encodeURI – кодирует URL-адрес целиком.
    decodeURI – декодирует URL-адрес целиком.
    encodeURIComponent – кодирует компонент URL, например, параметр, хеш, имя пути и т.п.
    decodeURIComponent – декодирует компонент URL.

Для URL целиком можно использовать encodeURI
Для параметров лучше будет взять encodeURIComponent
