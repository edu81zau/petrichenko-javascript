Сделйть слайдер. Ссылка на пример:
https://smartslider3.com/full-width-product-carousel/

Вариации слайдера:
1. Каждый элемент слайдера является div в контейнере слайдера.
Элементы статично прописаны уже в загруженной html странице. 
Кнопки создаются динамические, анализируя html

2. В html странице есть только контейнер для слайдера. 
При загрузке страницы вызывать конструктов слайдера, передавая ему опции. 
В опциях список параметров для каждого слайда. 
Дополнительно конфигурация для слайдера. (Скорость передвижения слайдов, количество активных слайдов,
положение точек для переключения)
Кнопки создаются динамические, на основании config, переданного конструктору

javascript создает контейнер для кнопок и назначает им класс. Внешний вид в отдельном css
